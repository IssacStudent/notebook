# Web以及网络基础

HTTP（HyperText Transfer Protocol，超文本传输协议）

__客户端__借助该协议发请求到__服务端__，获取**信息**

TCP/IP协议族：

1.   应用层（FTP, DNS）
2.   传输层（TCP, UDP）
3.   网络层（IP）（路由，选择数据传输路线）
4.   数据链路层
